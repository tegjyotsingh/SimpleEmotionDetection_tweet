This folder contains all the data generated during the process.

> Initia Sets: This folder contains data from the initial phases of the project: The original data of each hashtags, collected, preprocessed data, data after removing emoticons, etc.

> Experimenting_files: These are the sampled data whcih were used to test various classifiers in the intitial phases of the project.

> Weka_compatible: these are the csv and arff files used with weka for experimentation.

> Final_data: These are the final datasets with which all the full_data experimentations were done.

This folder containt the classifier files which use classifiers from pattern.vector.

> bayes.py, knn,py,svm,py : they implement the Naive Bayes, KNN and SVM classifiers respectively.

> Positive_pr, Negative_pr: these are the input preprocessed files. 

> test_20: this is a sample test file used by each of the classifiers, to depict what kind of output is made.
